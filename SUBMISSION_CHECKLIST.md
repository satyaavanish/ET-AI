- [ ] Rename the ZIP only if the portal requires a naming convention.

- [ ] Run `python -m pytest -q` once on the submission machine.
- [ ] Run the application and click **Run emergency demo**.
- [ ] Confirm B1 becomes critical and the response timeline appears.
- [ ] Confirm `/api/docs` loads.
- [ ] Record a 60–90 second backup demo video.
- [ ] Include `docs/architecture.svg` in the presentation.
- [ ] Use `docs/demo_script.md` for the live walkthrough.
- [ ] State prototype limitations before judges ask.
- [ ] Do not claim the paraphrased demo clauses are legal advice.
