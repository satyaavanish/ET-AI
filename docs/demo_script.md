# ZH-1 — Three-Minute Live Demo Script

## 0:00–0:20 — Frame the problem

“Plants already have sensors, permits, maintenance systems and worker rosters. The gap
is that these systems rarely reason together at the exact moment a decision is needed.”

## 0:20–0:45 — Show normal operations

Point to the zone heatmap, active permits and live KPIs. Explain that every API poll
creates a time-aligned frame across five signal types.

## 0:45–1:25 — Trigger the compound event

Click **Run emergency demo**. In B1, explain:

- CO is warning-level, not critical;
- hot work is active;
- maintenance is in progress;
- six workers are present.

Show the risk feed's two bars. The sensor-only counterfactual stays below alert while
the fused score crosses critical. This is the main differentiator.

## 1:25–1:55 — Show coordinated response

Walk through the response timeline: trigger, evacuate, notify, isolate work, preserve
evidence and draft report. Mention that each action is timestamped and exportable.

## 1:55–2:25 — Show continuous compliance

Show the hot-work gap in the compliance panel. Emphasise that the check runs against
live state instead of being performed only during a later audit.

## 2:25–2:50 — Search prior evidence

Search “hot work near elevated gas”. Explain that the retrieval layer is fully offline,
auditable and produces matched terms plus relevance scores.

## 2:50–3:00 — Close

“ZH-1 does not replace a safety officer. It gives the safety officer one fused,
explainable and actionable picture before isolated warning signals become an incident.”
